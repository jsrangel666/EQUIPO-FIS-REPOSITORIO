## ◈ SOBRE EL PROYECTO

**LUMEN** es una aplicación interactiva orientada a estudiantes universitarios, diseñada para la **organización visual de ideas, metas y procesos creativos–académicos**.

El sistema prioriza la experiencia del usuario y la introspección guiada, integrando elementos visuales y modos no tradicionales de interacción — incluyendo una identidad de usuario definida por un **gesto dibujado**.

> *"Ilumina tu proceso. Mapea tu mente."*

## ◈ FUNCIONALIDADES

| ID | Requerimiento |
|---|---|
| `RF-01` | Crear identidad mediante un **gesto dibujado** |
| `RF-02` | Validar el gesto para acceso al sistema |
| `RF-03` | Mostrar un **mapa visual interactivo** |
| `RF-04` | Crear nodos en el mapa |
| `RF-05` | Editar y eliminar nodos |
| `RF-06` | Conectar nodos entre sí |
| `RF-07` | Agregar texto descriptivo a cada nodo |
| `RF-08` | Guardar información del usuario |
| `RF-09` | Cargar información previamente guardada |
